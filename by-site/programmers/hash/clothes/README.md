# PRO 의상

- 티어: Level 2
- 유형 및 난이도(체감): 해시 / 쉬움
- 아이디어: 벗는 것도 하나의 의상으로 생각하여 (종류별 의상 수 + 1)을 모두 곱한 뒤, 아무것도 입지 않은 경우 1을 뺀다.
- 풀이 일자: 2025-12-23

## 회고

- 문제 해결의 핵심: 처음에는 단순히 의상의 개수를 세는 방향으로 접근했으나, 각 의상 종류별로 '입지 않는 경우'를 하나의 선택지로 포함해야 한다는 점이 가장 중요했다.

- Go의 맵(Map) 활용: Go 언어의 map[string]int를 사용하여 의상 종류별 카운트를 간결하게 구현할 수 있었다. 특히 존재하지 않는 키에 대해 ++ 연산을 수행하면 자동으로 기본값 0에서 시작하는 Go의 특성 덕분에 코드가 더 깔끔해졌다.

- 조합 공식의 유도: 모든 종류의 의상에 대해 (해당 종류의 의상 수 + 1)을 곱한 뒤, 모든 종류를 하나도 입지 않은 경우인 1을 빼주는 수학적 접근 방식을 익힐 수 있었다.
