- 코인 체인지가 항상 그리디로 풀리는 건 아니다(비정규 화폐체계). 하지만 본 문제는 배수 구조라 안전하다.

- 큰 동전부터 나눗셈 → 나머지 갱신 패턴을 익혀두면 유사 문제에서 바로 적용 가능.
